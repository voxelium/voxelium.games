Request User Data Deletion

When you install and use Voxelium Games mobile applications, we can collect and process some of your data for different legitimate purposes. You will find below explanations regarding the reasons why we may collect data.

Why is data collected?

Voxelium Games and its third-party partners collect data:
To provide you with a workable application.
To run analytics and understand how users interact with our app in order to continuously improve it.

The third-party partners that Voxelium Games will share the data with are:

  • GameAnalytics
  • Facebook Analytics
  • Adjust
  • Voodoo

Data deletion requests

To delete user data collected by Voxelium Games, please reach out to voxelium.games@gmail.com
